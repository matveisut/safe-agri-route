# Документы диплома — SafeAgriRoute

Краткая навигация по содержимому **этой** папки `diplom/` внутри репозитория (в [Obsidian](https://obsidian.md/) её можно открыть как корень хранилища — путь на диске может называться иначе).

Код приложения — в **корне репозитория** `safe-agri-route` (рядом с папкой `diplom/`).

| Файл | Назначение |
|------|------------|
| [ТЗ.md](ТЗ.md) | Спецификация MVP, пути API, сценарии демо; **§10** — целевая модель динамической угрозы (GNSS, IMU, радиоканал, рой, SDR, fusion) |
| [План обдуманный.md](План%20обдуманный.md) | Черновик раздела 1 ВКР + приложение «состояние кода» |
| [roadmap_task.md](roadmap_task.md) | Календарь, Гант, недельный план, вехи, риски |
| [zadanie_vkrs.md](zadanie_vkrs.md) | Формальное задание на ВКР (оба раздела, два студента) |
| [Промпты.md](Промпты.md) | Промпт **8** — симуляция §1.5 (`backend/simulation/`); промпты **9–12** — надстройка по [`ТЗ.md`](ТЗ.md) §10 (**реализована** в репозитории: признаки телеметрии, fusion, авто-replan, UI) |
| [Промпты_архив.md](Промпты_архив.md) | Полный архив промптов 1–7 (история) |
| [knowledge/](knowledge/) | Техническая документация (API, архитектура) — **без переноса**, на месте |

`ТЗ.md` **v1.3** (согласование §1/§10/§11 с реализацией fusion) и `План обдуманный.md` (v1.2) согласованы с репозиторием по состоянию на апрель 2026. **§10** в коде закрыт **эвристической** цепочкой (промпты 9–12): без отдельных драйверов SDR и без ML; расширения — в перспективе и в формулировках ТЗ.
