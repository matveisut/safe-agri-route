# О проекте (Project Context)
Проект "SafeAgriRoute" — это SaaS-платформа (MVP) для автоматизированного планирования безопасных маршрутов для групп сельскохозяйственных дронов с учетом киберугроз (GPS-спуфинг, глушение сигналов). 

# Технологический стек (Tech Stack)
## Backend
- Python 3.12
- FastAPI
- SQLAlchemy (Async)
- PostgreSQL + расширение PostGIS (для хранения геоданных)
- PyJWT (аутентификация)
- Google OR-Tools (решение задачи маршрутизации VRP)
- NetworkX (построение графов и расчет весов)

## Frontend
- React 18+ (сборщик Vite)
- TypeScript
- Tailwind CSS (стилизация)
- Zustand (управление состоянием)
- React-Leaflet + Leaflet.js (интерактивные карты)
- Axios (HTTP клиент)

# Архитектурные правила (Core Rules)
1. Строгая типизация: Используй Pydantic v2 для FastAPI и TypeScript интерфейсы для React.
2. Геоданные: Все координаты передаются и хранятся в формате WGS 84 (EPSG:4326). Для передачи полигонов полей на фронтенд используй формат GeoJSON.
3. Код-стайл: Пиши чистый, модульный код с подробными docstrings. Отделяй бизнес-логику (роутинг, математику) от контроллеров (API endpoints).
4. Заглушки (Mocks): Никаких бесконечных `// TODO`. Если нужно смоделировать полет дрона, пиши реальный генератор координат в памяти сервера, а не оставляй пустую функцию.
