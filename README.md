# SafeAgriRoute

**Отказоустойчивая система маршрутизации сельскохозяйственных БПЛА**  
*MVP-решение для планирования облетов полей в условиях радиоэлектронного противодействия (GPS-spoofing, Jamming).*

![Project Banner](https://img.shields.io/badge/Status-Beta-success) ![Stack](https://img.shields.io/badge/Stack-React%20%7C%20FastAPI%20%7C%20PostGIS-blue)

## 📌 О проекте
SafeAgriRoute решает проблему безопасного использования агродронов в зонах с возможными кибер-рисками. Обычные алгоритмы строят прямые пути, что может привести к потере дрона (в зоне глушения) или перехвату управления (в зоне спуфинга). 

Наше решение использует гео-пространственный анализ, кластеризацию и солверы Vehicle Routing Problem (CVRP) от Google OR-Tools, чтобы:
- Интеллектуально делить поле на непересекающиеся сектора (по числу дронов).
- Строить траектории в абсолютный обход кибер-угроз.
- Симулировать полет Роя (Swarm) в реальном времени через WebSockets.

## 🛠 Технологический стек
- **Frontend**: React 18, Vite, TypeScript, TailwindCSS, Zustand (стейт-менеджмент), React-Leaflet (карта).
- **Backend**: Python 3.12, FastAPI, SQLAlchemy (Async), GeoAlchemy2, NetworkX (графы), Google OR-Tools.
- **Инфраструктура / База данных**: Docker, PostgreSQL 15 + расширение PostGIS 3.3.

## 🚀 Установка и запуск

Проект разделен на два независимых слоя: инфраструктура БД/Бэкенда через Docker и Фронтенд на Node.js.

### 1. Подъем Бэкенда и Базы Данных (Docker)
Обязательно убедитесь, что установлен Docker Desktop / Engine.

```bash
# 1. Собрать образы и поднять контейнеры в фоне
docker-compose up -d --build

# 2. Инициализировать базу данных демо-информацией (Поле, Зоны угроз, Дроны)
docker-compose exec backend python seed.py
```
> Бэкенд будет крутиться по адресу: `http://localhost:8000`

### 2. Запуск Фронтенда
Требуется локальный NodeJS (v18+).

```bash
# Перейдите в папку фронтенда
cd frontend

# Установите npm-пакеты
npm install

# Запустите dev-сервер Vite
npm run dev
```
> Приложение откроется в браузере: `http://localhost:5173/`

## 🎮 Как пользоваться веб-интерфейсом
1. Зайдя в приложение, вы увидите "Пульт Управления" (левая колонка `MissionPanel`) и интерактивную Карту.
2. В блоке **Setup Mission** выберите Поле (Stavropol Krai) и отметьте галочками нужные дроны (разная мощность батареи).
3. Нажмите кнопку **Generate Neural Route**.
   - *Под капотом бэкенд сделает K-Means кластеризацию и решит задачу CVRP, избегая попадания в красные/желтые зоны.*
   - *На карте отрисуются цветные пунктирные пути.*
4. Нажмите **Start Telemetry Sim**.
   - *Откроется WebSocket, и симуляторы дронов (цветные маркеры) с большой скоростью полетят по координатам.*
